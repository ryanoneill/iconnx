# WS-3.5: BFloat16 (BF16) Half-Precision Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add BFloat16 as a first-class dtype variant alongside Float16, enabling end-to-end inference for BF16 ONNX models on the same surfaces FP16 just lit up. Land BERT-base BF16 (matmul-only canary) + Whisper-Tiny BF16 (Conv1D + attention + Cast-at-boundary stress) end-to-end through iconnx with ORT-matched outputs at empirically-measured tolerance.

**Architecture:** Strict additive-dispatch-arm architecture — BF16 is a new dtype variant added everywhere FP16 already exists. `Tensor::BFloat16(ArrayD<half::bf16>)` mirrors `Tensor::Float16`; CUDA-side `__nv_bfloat16` (from `<cuda_bf16.h>`) is bit-compatible with `half::bf16` (both `repr(transparent)` over a 16-bit word). Five compressed milestones (Q2-confirmed cadence): Y(1) IR + parser + dispatch infra (incl. `sm_80+` hardware check at `GpuTensor::zeros_bf16`); Y(2) kernel dispatch-arm wave (R3 wildcard removal + dtype-pair manual checklist); Y(3) MatMul + Conv via garboard `gemm_bf16` (im2col + GEMM mirror of M3.6); Y(4) BERT-base BF16 e2e; Y(5) Whisper-Tiny BF16 e2e + roster regression sweep.

**Tech Stack:** Rust 1.95 (matches CI), `half` crate (`half::bf16` for IEEE bfloat16; bit-compatible with CUDA `__nv_bfloat16`), `ndarray` (BF16 via `ArrayD<half::bf16>`), `prost` (ONNX protobuf — BFLOAT16 is `data_type = 16`), `garboard` (CUDA bindings; `gemm_bf16` already on main, single-flavor CUBLAS_COMPUTE_32F per `garboard/src/cuda/blas.rs:504`), CUDA `<cuda_bf16.h>` for NVRTC kernel intrinsics (`__bfloat162float`, `__float2bfloat16_rn`, `__hadd` on `__nv_bfloat16`).

```rust
// BFLOAT16 (data_type = 16) — added in WS-3.5. Parses parallel to
// FLOAT16 (data_type = 10) but via half::bf16 instead of half::f16.
16 => {
    let total_elements: usize = if dims.is_empty() { 1 } else { dims.iter().product() };
    if total_elements == 0 {
        return Ok(Tensor::from_vec_bf16(Vec::new(), dims));
    }
    if let Some(raw) = &tensor_proto.raw_data {
        if !raw.is_empty() {
            if raw.len() % 2 != 0 {
                return Err(ParseError::MalformedInitializer {
                    name: name.clone(),
                    dtype: "bfloat16",
                    reason: "raw_data length not multiple of 2".to_string(),
                });
            }
            let vals: Vec<half::bf16> = raw
                .chunks_exact(2)
                .map(|chunk| half::bf16::from_le_bytes([chunk[0], chunk[1]]))
                .collect();
            return Ok(Tensor::from_vec_bf16(vals, dims));
        }
    }
    if !tensor_proto.int32_data.is_empty() {
        // ONNX packs BF16 the same way it packs FP16: low 16 bits = bf16 bit pattern.
        let vals: Vec<half::bf16> = tensor_proto
            .int32_data
            .iter()
            .map(|v| half::bf16::from_bits(*v as u16))
            .collect();
        return Ok(Tensor::from_vec_bf16(vals, dims));
    }
    return Err(ParseError::MalformedInitializer {
        name: name.clone(),
        dtype: "bfloat16",
        reason: "no raw_data and no int32_data".to_string(),
    });
}
```

```rust
/// Create a new BFloat16 GPU tensor from shape and GPU data.
pub fn new_bf16(data: DeviceSlice<'static, half::bf16>, shape: Vec<usize>) -> Self {
    GpuTensor::BFloat16 {
        data: Arc::new(data),
        shape,
    }
}

/// Create a BFloat16 GPU tensor from host data.
///
/// Hardware-capability gate: BF16 native compute requires sm_80+.
/// Returns `CudaError::UnsupportedHardware` on older devices.
pub fn from_host_bf16(
    ctx: &IconnxCudaContext,
    data: &[half::bf16],
    shape: Vec<usize>,
) -> Result<Self, CudaError> {
    Self::check_bf16_hardware(ctx)?;
    let len: usize = if shape.is_empty() { 1 } else { shape.iter().product() };
    if data.len() != len {
        return Err(CudaError::ShapeMismatch {
            data_len: data.len(),
            expected: len,
        });
    }
    let mut gpu_data = ctx.alloc_zeros::<half::bf16>(len)?;
    ctx.memcpy_htod(data, &mut gpu_data)?;
    Ok(GpuTensor::BFloat16 {
        data: Arc::new(gpu_data),
        shape,
    })
}

/// Create a zeroed BFloat16 GPU tensor with given shape.
///
/// Hardware-capability gate: see `from_host_bf16`.
pub fn zeros_bf16(
    ctx: &IconnxCudaContext,
    shape: Vec<usize>,
) -> Result<Self, CudaError> {
    Self::check_bf16_hardware(ctx)?;
    let len: usize = if shape.is_empty() { 1 } else { shape.iter().product() };
    let gpu_data = ctx.alloc_zeros::<half::bf16>(len)?;
    Ok(GpuTensor::BFloat16 {
        data: Arc::new(gpu_data),
        shape,
    })
}

/// Copy BFloat16 tensor data back to host.
pub fn to_host_bf16(&self, ctx: &IconnxCudaContext) -> Result<Vec<half::bf16>, CudaError> {
    match self {
        GpuTensor::BFloat16 { data, shape } => {
            let len: usize = if shape.is_empty() { 1 } else { shape.iter().product() };
            let mut host = vec![half::bf16::from_f32(0.0); len];
            ctx.memcpy_dtoh(data, &mut host)?;
            Ok(host)
        }
        _ => Err(CudaError::DtypeMismatch {
            expected: "bfloat16",
            actual: self.dtype().name(),
        }),
    }
}

/// Hardware-capability check for BF16 native compute.
///
/// BF16 native compute requires sm_80+ (Ampere and later: A100, H100,
/// RTX 30/40-series). Older Turing (sm_75) compiles bf16 NVRTC but
/// emulates with significant perf loss; some intrinsics (`__bfloat162float`,
/// `__hadd` on `__nv_bfloat16`) require sm_80+ outright.
///
/// One-time-per-graph-load check (called from `zeros_bf16` /
/// `from_host_bf16`). Downstream dispatch sites can assume any
/// `GpuTensor::BFloat16` they receive is hardware-capable.
fn check_bf16_hardware(ctx: &IconnxCudaContext) -> Result<(), CudaError> {
    let (major, minor) = ctx.compute_capability();
    if major < 8 {
        return Err(CudaError::UnsupportedHardware {
            dtype: "bfloat16",
            required_compute_capability: "sm_80",
            device_compute_capability: format!("sm_{major}{minor}"),
        });
    }
    Ok(())
}
```

```rust
// WS-3.5 Y(2) sub-2a Transpose (BFloat16). Mirrors `transpose_general_kernel`
// in `kernels.rs` byte-for-byte with `unsigned short*` data pointers.
// ============================================================================
extern "C" __global__ void transpose_general_bf16_kernel(
    unsigned short* out, const unsigned short* inp,
    const size_t* in_shape, const size_t* in_strides,
    const size_t* out_strides, const size_t* perm,
    size_t ndim, size_t total_elements
) {
    size_t out_idx = blockIdx.x * blockDim.x + threadIdx.x;
    if (out_idx >= total_elements) return;
    size_t out_coords[6];
    size_t remaining = out_idx;
    for (size_t d = 0; d < ndim; d++) {
        out_coords[d] = remaining / out_strides[d];
        remaining = remaining % out_strides[d];
    }
    size_t in_idx = 0;
    for (size_t d = 0; d < ndim; d++) {
        in_idx += out_coords[d] * in_strides[perm[d]];
    }
    out[out_idx] = inp[in_idx];
}

// ============================================================================
extern "C" __global__ void concat_bf16_kernel(
    unsigned short* out, const unsigned short* inp,
    size_t inp_offset, size_t inp_axis_size,
    size_t outer_size, size_t inner_size,
    size_t out_axis_size, size_t total_inp_elements
) {
    size_t idx = blockIdx.x * blockDim.x + threadIdx.x;
    if (idx >= total_inp_elements) return;
    size_t outer = idx / (inp_axis_size * inner_size);
    size_t rem = idx % (inp_axis_size * inner_size);
    size_t axis_pos = rem / inner_size;
    size_t inner = rem % inner_size;
    size_t out_axis_pos = inp_offset + axis_pos;
    size_t out_idx = outer * (out_axis_size * inner_size) + out_axis_pos * inner_size + inner;
    out[out_idx] = inp[idx];
}

    unsigned short* out, const unsigned short* inp, const long long* indices,
    size_t num_indices, size_t slice_size, size_t dim0_size
) {
    size_t idx = blockIdx.x * blockDim.x + threadIdx.x;
    if (idx >= num_indices * slice_size) return;
    size_t which_index = idx / slice_size;
    size_t offset_in_slice = idx % slice_size;

    } else if ((size_t)adjusted >= dim0_size) {
        actual_index = dim0_size - 1;
    } else {

    }
```

```c
// ============================================================================
// WS-3.5 Y(2) sub-2b Elementwise BF16 kernels.
//
// __nv_bfloat16 supports __hadd/__hsub/__hmul/__hdiv on sm_80+ via
// <cuda_bf16.h> intrinsics. Pow/Sqrt/Erf go through f32 because there's
// no native bf16 transcendental — same pattern WS-3 used for FP16
// (no __half erf intrinsic existed either).
// ============================================================================

extern "C" __global__ void add_bf16_kernel(__nv_bfloat16* out, const __nv_bfloat16* a, const __nv_bfloat16* b, size_t n) {
    size_t i = blockIdx.x * blockDim.x + threadIdx.x;
    if (i < n) out[i] = __hadd(a[i], b[i]);
}

extern "C" __global__ void sub_bf16_kernel(__nv_bfloat16* out, const __nv_bfloat16* a, const __nv_bfloat16* b, size_t n) {
    size_t i = blockIdx.x * blockDim.x + threadIdx.x;
    if (i < n) out[i] = __hsub(a[i], b[i]);
}

extern "C" __global__ void mul_bf16_kernel(__nv_bfloat16* out, const __nv_bfloat16* a, const __nv_bfloat16* b, size_t n) {
    size_t i = blockIdx.x * blockDim.x + threadIdx.x;
    if (i < n) out[i] = __hmul(a[i], b[i]);
}

extern "C" __global__ void div_bf16_kernel(__nv_bfloat16* out, const __nv_bfloat16* a, const __nv_bfloat16* b, size_t n) {
    size_t i = blockIdx.x * blockDim.x + threadIdx.x;
    if (i < n) out[i] = __hdiv(a[i], b[i]);
}

extern "C" __global__ void pow_bf16_kernel(__nv_bfloat16* out, const __nv_bfloat16* base, const __nv_bfloat16* exp, size_t n) {
    size_t i = blockIdx.x * blockDim.x + threadIdx.x;
    if (i < n) {
        float b = __bfloat162float(base[i]);
        float e = __bfloat162float(exp[i]);
        out[i] = __float2bfloat16_rn(powf(b, e));
    }
}

extern "C" __global__ void sqrt_bf16_kernel(__nv_bfloat16* out, const __nv_bfloat16* x, size_t n) {
    size_t i = blockIdx.x * blockDim.x + threadIdx.x;
    if (i < n) out[i] = __float2bfloat16_rn(sqrtf(__bfloat162float(x[i])));
}

extern "C" __global__ void erf_bf16_kernel(__nv_bfloat16* out, const __nv_bfloat16* x, size_t n) {
    // BF16 has no native erf intrinsic — go through f32 (same pattern WS-3 used for FP16).
    size_t i = blockIdx.x * blockDim.x + threadIdx.x;
    if (i < n) out[i] = __float2bfloat16_rn(erff(__bfloat162float(x[i])));
}
```

### Step 10: Reduction BF16 with f32 accumulator (sub-commit 2c)

Reductions (Softmax, ReduceMean, LayerNorm) use f32 accumulator pattern. Add BF16 variants in `src/cuda/reduction.rs`. Each kernel reads BF16, promotes to f32 via `__bfloat162float` for accumulation, reduces, writes back via `__float2bfloat16_rn`.

```c
// ============================================================================
// WS-3.5 Y(2) sub-2c Softmax (BFloat16) — f32 accumulator.
// ============================================================================
extern "C" __global__ void softmax_bf16_kernel(
    __nv_bfloat16* out, const __nv_bfloat16* x,
    size_t outer_size, size_t axis_size, size_t inner_size
) {
    size_t outer = blockIdx.x;
    size_t inner = blockIdx.y;
    if (outer >= outer_size || inner >= inner_size) return;

    // f32 accumulator: max-reduce, exp-sum, then normalize.
    float max_v = -INFINITY;
    for (size_t a = 0; a < axis_size; a++) {
        size_t idx = outer * axis_size * inner_size + a * inner_size + inner;
        float v = __bfloat162float(x[idx]);
        if (v > max_v) max_v = v;
    }
    float sum = 0.0f;
    for (size_t a = 0; a < axis_size; a++) {
        size_t idx = outer * axis_size * inner_size + a * inner_size + inner;
        sum += expf(__bfloat162float(x[idx]) - max_v);
    }
    for (size_t a = 0; a < axis_size; a++) {
        size_t idx = outer * axis_size * inner_size + a * inner_size + inner;
        out[idx] = __float2bfloat16_rn(expf(__bfloat162float(x[idx]) - max_v) / sum);
    }
}

// ============================================================================
// WS-3.5 Y(2) sub-2c ReduceMean (BFloat16) — f32 accumulator.
// ============================================================================
extern "C" __global__ void reduce_mean_bf16_kernel(
    __nv_bfloat16* out, const __nv_bfloat16* x,
    size_t outer_size, size_t reduce_size, size_t inner_size
) {
    size_t outer = blockIdx.x * blockDim.x + threadIdx.x;
    size_t inner = blockIdx.y * blockDim.y + threadIdx.y;
    if (outer >= outer_size || inner >= inner_size) return;

    float sum = 0.0f;
    for (size_t r = 0; r < reduce_size; r++) {
        size_t idx = outer * reduce_size * inner_size + r * inner_size + inner;
        sum += __bfloat162float(x[idx]);
    }
    float mean = sum / (float)reduce_size;
    out[outer * inner_size + inner] = __float2bfloat16_rn(mean);
}

// ============================================================================
// WS-3.5 Y(2) sub-2c LayerNorm (BFloat16) — f32 accumulator.
// (Inputs/outputs BF16; mean/var/scale/bias in f32 internally.)
// ============================================================================
extern "C" __global__ void layer_norm_bf16_kernel(
    __nv_bfloat16* out, const __nv_bfloat16* x,
    const float* scale, const float* bias,
    size_t batch_size, size_t feature_size, float eps
) {
    size_t b = blockIdx.x * blockDim.x + threadIdx.x;
    if (b >= batch_size) return;

    float mean = 0.0f;
    for (size_t f = 0; f < feature_size; f++) {
        mean += __bfloat162float(x[b * feature_size + f]);
    }
    mean /= (float)feature_size;

    float var = 0.0f;
    for (size_t f = 0; f < feature_size; f++) {
        float d = __bfloat162float(x[b * feature_size + f]) - mean;
        var += d * d;
    }
    var /= (float)feature_size;
    float inv_std = rsqrtf(var + eps);

    for (size_t f = 0; f < feature_size; f++) {
        size_t idx = b * feature_size + f;
        float normalized = (__bfloat162float(x[idx]) - mean) * inv_std;
        float scaled = normalized * scale[f] + bias[f];
        out[idx] = __float2bfloat16_rn(scaled);
    }
}
```

```rust
fn box_muller_seeded_mel(n: usize, seed: u64) -> Vec<f32> {
    // xorshift64* PRNG → Box-Muller transform → unit-Gaussian samples.
    let mut state = seed;
    let mut next_u64 = || -> u64 {
        state ^= state >> 12;
        state ^= state << 25;
        state ^= state >> 27;
        state.wrapping_mul(0x2545F4914F6CDD1D)
    };
    let mut next_uniform = || -> f32 {
        // Map u64 to (0, 1).
        ((next_u64() >> 11) as f32) / ((1u64 << 53) as f32)
    };
    let mut out = Vec::with_capacity(n);
    while out.len() < n {
        let u1 = next_uniform().max(1e-9); // avoid log(0)
        let u2 = next_uniform();
        let r = (-2.0_f32 * u1.ln()).sqrt();
        let theta = 2.0_f32 * std::f32::consts::PI * u2;
        out.push(r * theta.cos());
        if out.len() < n {
            out.push(r * theta.sin());
        }
    }
    out
}
```
